# 360° Retail Performance & Customer Insights Dashboard

## Project Overview

This project demonstrates the complete data analysis workflow using an online retail dataset. The objective was to clean transaction data, calculate key performance indicators (KPIs), segment customers using RFM analysis, and build an interactive dashboard to support business decision-making.

## What I Did

* Imported retail transaction data using Power Query
* Cleaned data by removing cancelled transactions and handling missing values
* Created calculated columns such as Revenue, Year, Month, and Year-Month
* Calculated key metrics including Total Revenue, Total Orders, Average Order Value (AOV), and Return Rate
* Performed **RFM (Recency, Frequency, Monetary) customer segmentation**
* Built an interactive dashboard with charts and slicers
* Created a What-If analysis to evaluate the impact of discount changes on profit

## How I Did It

* Used Microsoft Excel with Power Query for data preparation
* Applied formulas and PivotTables to compute KPIs and customer metrics
* Used functions like SUM, XLOOKUP, and QUARTILE to analyze customer behavior
* Designed visualizations such as line charts, bar charts, and stacked charts
* Implemented Data Table (What-If Analysis) to simulate profit scenarios

## Result

* Identified monthly revenue trends and top-performing products
* Highlighted countries contributing the highest sales
* Segmented customers into value-based groups using RFM analysis
* Evaluated how discount changes affect overall profit
* Built a professional, interactive retail performance dashboard

## Tools Used

* Microsoft Excel
* Power Query
* PivotTables
* Data Visualization
